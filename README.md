# 🌙 שינה של התינוק

אפליקציית ניהול לו"ז שינה לתינוק. נבנתה כ-PWA שניתן להתקין על מסך הבית של האייפון.

## הרצה מקומית

```bash
npm install
npm run dev
```

פתחי את http://localhost:3000

## דיפלוי ל-Vercel (מומלץ, חינם)

### אפשרות 1 – דרך הדפדפן
1. העלי את הפרויקט ל-GitHub
2. היכנסי ל-vercel.com עם חשבון GitHub
3. לחצי "New Project" ובחרי את הריפו
4. לחצי Deploy — זהו!

### אפשרות 2 – דרך הטרמינל
```bash
npm install -g vercel
vercel --prod
```

## התקנה על האייפון (PWA)
1. פתחי את הכתובת בספארי
2. לחצי על כפתור השיתוף ↑
3. בחרי "הוסף למסך הבית"
4. האפליקציה תופיע כאייקון!

## נתונים
הנתונים נשמרים ב-localStorage – פרטי ומאובטח, לא עולה לאינטרנט.
